## 🚀 Auto-Attacher (CE-AutoAttacher.exe)

The included `CE-AutoAttacher.exe` simplifies everything:

1. Run `CE-AutoAttacher.exe` **as Administrator**.
2. Select your Cheat Engine installation path (or let it auto-detect).
3. Choose which cheats you want to enable.
4. Click "Attach" – the tool will launch Cheat Engine, load the table, and attach to the game automatically.
5. All cheats are activated instantly.

> ℹ️ The auto-attacher is statically compiled and password‑protected to avoid false positives. Add to exclusions if needed.

## ❗ Troubleshooting

| Problem | Solution |
|---------|----------|
| Game crashes after enabling cheats | Update REFramework to the latest version . |
| Table shows "??" values | Re-attach to the game process. |
| Auto-attacher flagged by antivirus | Temporarily disable AV – it's a false positive. |
| Cheats not working on GamePass | GamePass version uses a different process name. Try manual attachment. |
